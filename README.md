Скрипт, который решает мини-игру в Gothic 1 Remake при взломе замков.
Вводные данные необходимо записать в файл in.txt
Вводные данные имеют вид:
a = 1
b = 7
c = 6
d = 1
e = 7
f = 2
Где a,b,c,d,e,f - номера пластин, начиная от ближней. Значения - начальные номера пинов, считаются слева-направо от 1 до 7

aA -> bD, eA
bA -> 0
cA -> bD, aD
dA -> bA, fD
eA -> 0
fA -> eD
Где a,b,c,d,e,f - номера пластин, A и D - направление перемещения пластин (A - влево, D - вправо), aA, aD и т.д. - обозначают смещение пластины a..f в ту или иную сторону, bA -> 0 - обозначает то, что при перемещении пластины b другие пластины не перемещаются, aA -> bD, eA - обозначает то, что при перемещении пластины a влево, пластина b перемещается вправо, пластина e перемещается влево.
Перед запуском скрипта следует записать все начальные положения пинов и прокликать все пластины и записать их зависимости в файл in.txt
